html

html semantic elements

Tags Name Description
<article> Article A portion of the document that is self-contained
<aside> Aside A portion of the document that is indirectly related
<details> Details Creates a toggleable widget that contains info. must include the <summary> element
<figure> Figure A self-contained content like images and videos
<figcaption> Figure Caption A caption describing the content of the parent <figure> element
<nav> Navigation A portion that contains navigation links
<section> Section A generic standalone section of the document
<summary> Summary The text inside of a <details> element
